# treeos-base

**treeos-base** is the essential metapackage for TreeOS. It includes core utilities and essential packages to get TreeOS up and running.

## Included Packages

- **git**: Version control tool
- **neovim**: Text editor
- **zsh**: Z shell
- **powerlevel10k**: Zsh prompt theme
- **ufw**: Uncomplicated Firewall
- **base-devel**: Development tools

## Installation

To install the base package on TreeOS:

grow install treeos-base
